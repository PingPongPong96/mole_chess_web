UI 编辑器快速上手

3分钟起步

  1. 先切 `Surface`：`entry` 改首页、`index` 改对局。
  2. 点 `素材库` 上传图片，再在 `背景绑定` 里选 `entry/game/board`。
  3. 在 `事件列表` 选事件，右侧开启事件并新增条件/动作。
  4. 点 `应用` 看中间预览，满意后 `保存`。

条件编辑器（新版）

  1. 先选条件类型，再选操作符。
  2. `exists/not_exists` 不需要填值，会自动隐藏值输入。
  3. `dice` 自动使用数字输入与数值比较操作符。
  4. `in` 会变成标签式多值输入（不用逗号手写）。

推荐排错顺序

  1. 确认当前 `Surface` 对不对。
  2. 事件是否 `启用`，条件是否过严。
  3. 动作里素材是否存在（删除素材会导致引用失效）。
  4. 必要时 `刷新预览`，再看结果。
